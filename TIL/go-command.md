# Go Command

Go의 명령어에 대해서 간략하게 알아보자.  

```
go version
- go의 현재 설치된 버전
go help
- go 명령어에 대한 도움말 요약
go env
- go 관련 환경 변수 목록
go fmt
- 소스코드를 go스럽게 정렬
go run
- 소스코드를 빌드하고 실행 
go build
- 소스코드에 대한 실행 파일을 생성(현재위치)
go install
- 상위 디렉토리 전체를 대상으로(상위 디렉토리 이름으로) 한 실행 파일을 생성  
  ($GOPATH/bin 위치)
go get 
- module 생성 후에 필요한 종속 패키지들을 설치 또는 종속성 버전 업그레이드
```
